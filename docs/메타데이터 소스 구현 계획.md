# eztag 메타데이터 소스 구현 계획

> 작성일: 2026-03-09 · eztag v0.4.x 대상

---

## 1. 개요

Spotify 외에 **Bugs**, **Melon**, **Apple Music** 3개 소스를 추가 구현한다.
각 소스는 설정(Settings → 메타데이터 소스)에서 활성/비활성 전환이 가능하며,
태그 검색 다이얼로그에서 소스를 선택하여 선택적으로 검색할 수 있다.

### 구현 우선순위

| 순위 | 소스 | 방식 | 난이도 | 한국 음원 |
|-----|------|------|--------|-----------|
| 1 | **Bugs** | JSON API + HTML 스크래핑 | 낮음 | ✅ 최적 |
| 2 | **Apple Music** | HTML 파싱 (JSON-LD + serialized-server-data) | 중간 | ✅ 양호 |
| 3 | **Melon** | HTML 스크래핑 | 높음 (봇 방지 가능성) | ✅ 최적 |

## 2. 각 소스 기술 분석

### 2-1. Bugs

**검색 API** (JSON, 인증 불필요):
```
GET https://m.bugs.co.kr/api/getSearchList?type=album&query={query}&page=1&size=20
```

응답 구조:
```json
{
  "list": [
    {
      "album_id": 20032445,
      "title": "A New Trilogy",
      "artists": [{"artist_nm": "Lovelyz"}],
      "release_ymd": "20170222"
    }
  ]
}
```

**앨범 상세 페이지** (HTML 스크래핑):
```
GET https://music.bugs.co.kr/album/{album_id}
```

추출 필드:
- 앨범명: `<h1>` in `.pgTitle`
- 앨범 아티스트: `<table class="info">` → 아티스트 행
- 커버 URL: `<div class="photos"> > img src` (1000px 버전)
- 유형(album_type): 유형 행 `<td>`
- 장르: 장르 행 (멀티 링크)
- 발매일: `<time datetime="...">` YYYY.MM.DD
- 기획사(label): 기획사 행
- 유통사(distributor): 유통사 행
- 총 트랙수: `"desc">(N)</p>`
- 트랙 목록: `<tr albumId="...">` 반복
  - disc_no: `disc_id="..."` 속성
  - track_no: `<p class="trackIndex"> > <em>`
  - title: `title="..."` 속성 (삽입곡 설명 제거)
  - artist: `<p class="artist">`

**커버 URL 패턴**:
```python
album_id = 20032445
cover = f"https://image.bugsm.co.kr/album/images/1000/{str(album_id)[:-2]}/{album_id}.jpg"
# → https://image.bugsm.co.kr/album/images/1000/200324/20032445.jpg
```

**곡 검색 API** (JSON):
```
GET https://m.bugs.co.kr/api/getSearchList?type=track&query={query}&page=1&size=20
```

### 2-2. Apple Music

**검색 URL** (HTML 파싱, 인증 불필요):
```
GET https://music.apple.com/{storefront}/search?term={query}
```
- 한국: `kr` (스토어 ID: 143466)
- 미국: `us` (스토어 ID: 143441)

응답 HTML 내 `<script type="application/json" id="serialized-server-data">` 태그에 JSON 데이터 포함.

검색 결과 파싱 (JSON):
```
data.sections[].items[].contentDescriptor.{kind, url, artwork, title, trackCount, showExplicitBadge}
```
- `kind == "album"` → 앨범
- `kind == "song"` → 곡

**앨범 상세 URL** (HTML 파싱):
```
GET https://music.apple.com/{storefront}/album/{album_id}
```

동일 방식으로 `serialized-server-data` JSON + `<script id="schema:music-album" type="application/ld+json">` 파싱.

추출 필드:
- album: `json_select "title"`
- albumartist: `subtitleLinks[].title` (Various Artists 필터링)
- cover_url: `artwork.dictionary.url` → `{base}/{width}x{height}-{quality}.jpg`
- release_date: `datePublished` (schema ld+json) 또는 `seoData.musicReleaseDate`
- genre: `genre` 배열 (schema ld+json)
- total_tracks: `trackCount`
- label: `linkSection.link` 중 `kind == "recordLabel"`
- 트랙 목록:
  - title: `title`
  - track_no: `trackNumber`
  - disc_no: `discNumber`
  - composer: `composer`
  - isrc: `seoData.ogSongs[].attributes.isrc`
  - explicit: `showExplicitBadge` or `contentRating`

**인증 없이 사용 가능** — public HTML 페이지이므로 Bearer 토큰 불필요.
(amp-api.music.apple.com은 사용하지 않고 직접 HTML 파싱)

### 2-3. Melon

**검색 URL** (HTML 스크래핑):
```
GET https://www.melon.com/search/album/index.htm?section=album&searchGnbYn=Y&kkoSpl=N&kkoDpType=&sort=weight&q={query}
```

**앨범 상세 URL**:
```
GET https://www.melon.com/album/detail.htm?albumId={album_id}
```

추출 필드:
- album_id: `goAlbumDetail('...')` JS 콜백에서 추출
- 앨범명: `song_name` 클래스
- 앨범 아티스트: `artist` 클래스
- 커버 URL: `<meta property="og:image">` (1000px 버전으로 치환)
- 발매일: `<dt>발매일</dt>` 다음 `<dd>`
- 발행사(label): `<dt>발매사</dt>` 다음 `<dd>`
- 트랙 목록: `<tr data-group-items="...">` 반복
  - disc_no: `cd{N}` 추출
  - track_no: `<span class="rank">`
  - title: `wrap_song_info`
  - artist: `ellipsis rank02`

**주의사항**:
- User-Agent 헤더 필수: `Mozilla/5.0 (Windows NT 10.0; Win64; x64) ...`
- Referer 헤더 권장: `https://www.melon.com`
- 봇 방지 차단 시 `requests-html` 또는 `cloudscraper` 사용 고려
- Cookie: `PCID`, `PC_PCID` 등이 필요할 수 있음 → requests.Session() 활용

### 3-3. bugs.py 구현

```python
import requests
from bs4 import BeautifulSoup

SEARCH_API = "https://m.bugs.co.kr/api/getSearchList"
ALBUM_URL  = "https://music.bugs.co.kr/album/{album_id}"
COVER_URL  = "https://image.bugsm.co.kr/album/images/1000/{prefix}/{album_id}.jpg"

def search_albums(query: str, limit: int = 10) -> list[dict]:
    """앨범명 검색 → 기본 정보 반환"""
    resp = requests.get(SEARCH_API, params={"type":"album","query":query,"page":1,"size":limit}, timeout=10)
    data = resp.json()
    results = []
    for item in data.get("list", []):
        album_id = str(item["album_id"])
        results.append({
            "provider": "bugs",
            "type": "album",
            "provider_id": album_id,
            "title": item["title"],
            "artist": ", ".join(a["artist_nm"] for a in item.get("artists", [])),
            "release_date": _parse_ymd(item.get("release_ymd", "")),
            "cover_url": _cover_url(album_id),
        })
    return results

def get_album_tracks(album_id: str) -> dict:
    """앨범 상세 페이지 파싱 → 앨범 공통 태그 + 트랙 목록 반환"""
    resp = requests.get(ALBUM_URL.format(album_id=album_id), timeout=10,
                        headers={"User-Agent": "Mozilla/5.0"})
    soup = BeautifulSoup(resp.text, "html.parser")
    # ... HTML 파싱 로직
    return {"album": {...}, "tracks": [...]}

def _cover_url(album_id: str) -> str:
    prefix = album_id[:-2]
    return COVER_URL.format(prefix=prefix, album_id=album_id)

def _parse_ymd(ymd: str) -> str:
    """'20170222' → '2017-02-22'"""
    if len(ymd) == 8:
        return f"{ymd[:4]}-{ymd[4:6]}-{ymd[6:]}"
    return ymd
```

### 3-4. apple_music.py 구현

```python
import requests
import json
import re
from bs4 import BeautifulSoup

SEARCH_URL = "https://music.apple.com/{storefront}/search?term={query}"
ALBUM_URL  = "https://music.apple.com/{storefront}/album/{album_id}"

HEADERS = {
    "User-Agent": "Mozilla/5.0 (Windows NT 10.0; Win64; x64) AppleWebKit/537.36",
    "Accept-Language": "ko-KR,ko;q=0.9",
}

def search_albums(query: str, storefront: str = "kr", limit: int = 10) -> list[dict]:
    url = SEARCH_URL.format(storefront=storefront, query=requests.utils.quote(query))
    resp = requests.get(url, headers=HEADERS, timeout=15)
    return _parse_search_results(resp.text, result_type="album", limit=limit)

def _parse_search_results(html: str, result_type: str, limit: int) -> list[dict]:
    """serialized-server-data JSON 파싱"""
    match = re.search(
        r'<script type="application/json" id="serialized-server-data">\[(.+?)\]</script>',
        html, re.DOTALL
    )
    if not match:
        return []
    data = json.loads(match.group(1))
    results = []
    # data.sections[].items[]에서 album 또는 song 추출
    for section in data.get("data", {}).get("sections", []):
        for item in section.get("items", []):
            cd = item.get("contentDescriptor", {})
            if cd.get("kind") != result_type:
                continue
            # URL에서 album_id 추출
            url = cd.get("url", "")
            album_id = url.rstrip("/").split("/")[-1]
            artwork = cd.get("artwork", {}).get("dictionary", {})
            cover_base = re.sub(r'/[^/]+$', '', artwork.get("url", ""))
            results.append({
                "provider": "apple_music",
                "type": "album",
                "provider_id": album_id,
                "spotify_url": url,   # 실제로는 apple_music_url
                "title": cd.get("title", ""),
                "artist": ", ".join(l.get("title","") for l in cd.get("subtitleLinks",[])),
                "cover_url": f"{cover_base}/600x600-90.jpg" if cover_base else "",
                "total_tracks": cd.get("trackCount"),
                "explicit": bool(cd.get("showExplicitBadge")),
            })
            if len(results) >= limit:
                break
    return results

def get_album_tracks(album_id: str, storefront: str = "kr") -> dict:
    """앨범 상세 페이지 파싱"""
    url = ALBUM_URL.format(storefront=storefront, album_id=album_id)
    resp = requests.get(url, headers=HEADERS, timeout=15)
    return _parse_album_page(resp.text)

def _parse_album_page(html: str) -> dict:
    """schema:music-album (ld+json) + serialized-server-data 파싱"""
    # ld+json: datePublished, genre, tracks[].{name, isrc}
    # serialized-server-data: albumartist, label, trackCount, per-track info

```

def normalize_album(raw: dict, provider: str) -> dict:
    """provider별 raw 데이터를 eztag 공통 스키마로 변환"""
    return {
        "provider": provider,
        "type": "album",
        "provider_id": str(raw.get("provider_id", "")),
        "title": raw.get("title", "") or "",
        "artist": raw.get("artist", "") or "",
        "album_artist": raw.get("album_artist") or raw.get("artist", ""),
        "cover_url": raw.get("cover_url", ""),
        "release_date": raw.get("release_date", ""),
        "year": _extract_year(raw.get("release_date", "")),
