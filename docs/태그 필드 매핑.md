# eztag 태그 필드 매핑 참조

> 참조: [Mp3tag Format Strings](https://docs.mp3tag.de/format/) · [Mp3tag Tag Field Mapping](https://docs.mp3tag.de/mapping/)

## 1. eztag 내부 필드명 ↔ 오디오 포맷별 태그 매핑

eztag에서 사용하는 내부 필드명(Python/JS)과 각 오디오 포맷의 실제 태그 키를 정리합니다.

| eztag 필드명     | MP3 (ID3v2)       | FLAC / OGG (Vorbis) | M4A (MP4 iTunes)  | 지원 여부 |
|-----------------|-------------------|---------------------|-------------------|-----------|
| `title`         | `TIT2`            | `TITLE`             | `©nam`            | ✅ 읽기/쓰기 |
| `artist`        | `TPE1`            | `ARTIST`            | `©ART`            | ✅ 읽기/쓰기 |
| `album_artist`  | `TPE2`            | `ALBUMARTIST`       | `aART`            | ✅ 읽기/쓰기 |
| `album_title`   | `TALB`            | `ALBUM`             | `©alb`            | ✅ 읽기/쓰기 |
| `track_no`      | `TRCK` (N 또는 N/T) | `TRACKNUMBER`     | `trkn` (N, T)     | ✅ 읽기/쓰기 |
| `total_tracks`  | `TRCK` (N/T 형식) | `TRACKTOTAL`        | `trkn` tuple[1]   | ✅ 쓰기 전용 |
| `disc_no`       | `TPOS`            | `DISCNUMBER`        | `disk` (N, 0)     | ✅ 읽기/쓰기 |
| `year`          | `TDRC` / `TYER`   | `DATE`              | `©day` (앞 4자리) | ✅ 읽기/쓰기 |
| `release_date`  | `TDRC` (전체 날짜) | `DATE` (전체 날짜) | `©day` (전체)     | ✅ 쓰기 전용 |
| `genre`         | `TCON`            | `GENRE`             | `©gen`            | ✅ 읽기/쓰기 |
| `label`         | `TPUB`            | `ORGANIZATION`      | `©pub`            | ✅ 쓰기 전용 |
| `lyrics`        | `USLT`            | `LYRICS`            | `©lyr`            | ✅ 읽기/쓰기 |
| `has_cover`     | `APIC` (존재 여부) | pictures 배열       | `covr`            | ✅ 읽기 전용 |

### 읽기 전용 (Spotify 검색 시 표시, 파일에 저장 불가)

| eztag 필드명     | 설명                          | Spotify 제공 여부 |
|-----------------|-------------------------------|-------------------|
| `explicit`      | 19금 여부                     | ✅                |
| `popularity`    | 인기도 (0–100)                | ✅                |
| `isrc`          | 국제 표준 녹음 코드            | ✅ (트랙 검색)    |
| `album_type`    | album / single / compilation  | ✅                |
| `spotify_id`    | Spotify 트랙/앨범 ID          | ✅                |
| `spotify_url`   | Spotify 링크                  | ✅                |
| `preview_url`   | 30초 미리 듣기 URL            | ✅                |
| `duration`      | 재생 시간 (초)                | ✅ (Spotify 제공) |

## 2. 포맷별 상세 태그 키

### MP3 — ID3v2.3 / ID3v2.4

| 필드           | ID3 프레임 | 비고 |
|---------------|-----------|------|
| 제목           | `TIT2`    | |
| 아티스트       | `TPE1`    | |
| 앨범 아티스트   | `TPE2`    | |
| 앨범           | `TALB`    | |
| 트랙 번호      | `TRCK`    | `"N"` 또는 `"N/T"` 문자열 |
| 디스크 번호    | `TPOS`    | |
| 연도 (ID3v2.4) | `TDRC`    | 전체 날짜 가능: `YYYY-MM-DD` |
| 연도 (ID3v2.3) | `TYER`    | 4자리 연도만 |
| 장르           | `TCON`    | |
| 레이블/발행사   | `TPUB`    | |
| 가사 (비동기)  | `USLT`    | lang=`eng` |
| 커버아트       | `APIC`    | |
| 작곡가         | `TCOM`    | ⬜ 미지원 |
| BPM           | `TBPM`    | ⬜ 미지원 |
| 코멘트         | `COMM`    | ⬜ 미지원 |
| 정렬용 아티스트 | `TSOP`    | ⬜ 미지원 |

### FLAC / OGG — Vorbis Comment

| 필드           | Vorbis 키          | 비고 |
|---------------|--------------------|------|
| 제목           | `TITLE`            | |
| 아티스트       | `ARTIST`           | |
| 앨범 아티스트   | `ALBUMARTIST`      | 또는 `ALBUM ARTIST` |
| 앨범           | `ALBUM`            | |
| 트랙 번호      | `TRACKNUMBER`      | |
| 총 트랙 수     | `TRACKTOTAL`       | 또는 `TOTALTRACKS` |
| 디스크 번호    | `DISCNUMBER`       | |
| 날짜           | `DATE`             | `YYYY` 또는 `YYYY-MM-DD` |
| 장르           | `GENRE`            | |
| 레이블         | `ORGANIZATION`     | |
| 가사           | `LYRICS`           | |
| 작곡가         | `COMPOSER`         | ⬜ 미지원 |
| 코멘트         | `COMMENT`          | ⬜ 미지원 |

### M4A / AAC — iTunes MP4 Atoms

| 필드           | MP4 Atom   | 비고 |
|---------------|------------|------|

## 3. Mp3tag 포맷 스트링 ↔ eztag 필드명

Mp3tag 스타일의 `%field%` 포맷 변수와 eztag 내부 필드명 대응표입니다.
파일명 자동 생성, 가져오기/내보내기 등에 활용할 수 있습니다.

| Mp3tag 변수      | eztag 필드명     | 예시 값               |
|-----------------|-----------------|----------------------|
| `%title%`       | `title`         | `"Shape of You"`     |
| `%artist%`      | `artist`        | `"Ed Sheeran"`       |
| `%albumartist%` | `album_artist`  | `"Ed Sheeran"`       |
| `%album%`       | `album_title`   | `"÷ (Divide)"`       |
| `%track%`       | `track_no`      | `3`                  |
| `%totaltracks%` | `total_tracks`  | `16`                 |
| `%disc%`        | `disc_no`       | `1`                  |
| `%year%`        | `year`          | `2017`               |
| `%genre%`       | `genre`         | `"Pop"`              |
| `%comment%`     | *(미지원)*       | —                    |
| `%composer%`    | *(미지원)*       | —                    |
| `%publisher%`   | `label`         | `"Atlantic Records"` |
| `%lyrics%`      | `lyrics`        | 가사 텍스트          |
| `%_filename%`   | `filename`      | `"01 Shape of You.mp3"` |
| `%_extension%`  | (path에서 추출) | `"mp3"`              |
| `%_bitrate%`    | `bitrate`       | `320`                |
| `%_length%`     | `duration`      | `"3:54"`             |
| `%_codec%`      | `file_format`   | `"MP3"`              |
| `%_cover_size%` | (미저장)         | —                    |

### 파일명 포맷 패턴 예시

```
# 일반적인 트랙 파일명
%track%-%title%           →  03-Shape of You.mp3
%track%-%artist%-%title%  →  03-Ed Sheeran-Shape of You.mp3

# 앨범 폴더 구조
%albumartist%/%year% - %album%/%track% - %title%
→ Ed Sheeran/2017 - Divide/03 - Shape of You.mp3

# 디스크 포함
%disc%-%track%-%title%    →  1-03-Shape of You.mp3
```

## 4. Spotify API → eztag 필드 매핑

Spotify 검색 결과를 eztag 태그 필드로 변환하는 매핑표입니다.

### 앨범 공통 태그 (전체 파일 일괄 적용)

| Spotify 필드          | eztag 필드명     | 오디오 파일 저장 |
|----------------------|-----------------|-----------------|
| `album.artists[0]`   | `artist`        | ✅               |
| `album.artists[0]`   | `album_artist`  | ✅               |
| `album.name`         | `album_title`   | ✅               |
| `album.release_date` | `release_date`  | ✅ (전체 날짜)   |
| `album.release_date[:4]` | `year`     | ✅ (연도만)      |
| `album.total_tracks` | `total_tracks`  | ✅               |
| `artists[0].genres[0]` | `genre`      | ✅               |
| `label` (full album) | `label`         | ✅               |
| `images[0].url`      | `cover_url`     | ✅ (다운로드 저장) |
| `album_type`         | `album_type`    | ❌ 표시 전용     |
| `popularity`         | `popularity`    | ❌ 표시 전용     |
| `genres` (album)     | `genres`        | ❌ 표시 전용     |

### 트랙별 태그 (매칭된 파일에 개별 적용)

| Spotify 필드          | eztag 필드명  | 오디오 파일 저장 |
|----------------------|--------------|-----------------|
| `name`               | `title`      | ✅               |
| `track_number`       | `track_no`   | ✅               |
| `disc_number`        | `disc_no`    | ✅               |
| `artists[0].name`    | `artist`     | ✅ (앨범 아티스트와 다를 때만) |
| `external_ids.isrc`  | `isrc`       | ❌ 표시 전용     |
| `explicit`           | `explicit`   | ❌ 표시 전용     |
| `duration_ms`        | `duration`   | ❌ 표시 전용     |
| `preview_url`        | `preview_url`| ❌ 표시 전용     |

## 6. 태그 쓰기 구현 요약 (tag_writer.py)

```python
# eztag 내부 → 파일 저장 시 사용되는 실제 키

MP3 (ID3v2):
  title         → TIT2
  artist        → TPE1
  album_artist  → TPE2
  album_title   → TALB
  genre         → TCON
  label         → TPUB
  track_no      → TRCK  ("N" 또는 "N/T" 포함 total_tracks)
  disc_no       → TPOS
  release_date  → TDRC  (우선), year → TDRC (폴백)
  lyrics        → USLT  (lang=eng)

FLAC / OGG (Vorbis):
  title         → title / TITLE
  artist        → artist / ARTIST
  album_artist  → albumartist / ALBUMARTIST
  album_title   → album / ALBUM
  genre         → genre / GENRE
  label         → organization / ORGANIZATION
  track_no      → tracknumber / TRACKNUMBER
  total_tracks  → tracktotal / TRACKTOTAL
  disc_no       → discnumber / DISCNUMBER
  release_date  → date / DATE  (우선), year (폴백)
  lyrics        → lyrics / LYRICS

M4A (MP4 iTunes):
  title         → ©nam
  artist        → ©ART
  album_artist  → aART
  album_title   → ©alb
  genre         → ©gen
  label         → ©pub
  track_no      → trkn  tuple (track_no, total_tracks)
  disc_no       → disk  tuple (disc_no, 0)
  release_date  → ©day  (우선), year (폴백)
  lyrics        → ©lyr
```
