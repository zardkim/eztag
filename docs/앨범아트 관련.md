앨범아트 관련 기능

1. 앨범아트 추가, 추출, 제거, 수정 기능
2. 앨범아트 종류 설정기능
   - Front Cover
   - Back Cover
   - Other
   - Icon
   - Leaflet
   - Media
   - Lead Artist
   - Conductor
   - Band
   - Composer
   - Lyricist
   - Recording Location
   - During Recording
   - During Performance
   - Video Capture
   - illustration
   - Band Logotype
   - Publisher Logotype
3. 앨범아트 기본값은 Front Cover 로 설정. 기본 연결 파일명은 cover, front, front cover랑 연결. back, back cover는 Bakcer Cover랑 연결
4. 앨범아트 최대 이미지 크기는 1200x1200 으로 설정.
5. 앨범아트에 이미지 종류 및 용량, 크기, 앨범아트 종류를 표시하기
